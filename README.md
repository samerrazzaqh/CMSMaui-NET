# CMS Project with .NET Maui Blazor  
🚀 A practical **.NET Maui Blazor** application for building a full **CMS** project.  


## 📌 Description  
This project is a hands-on **.NET Maui Blazor** implementation of an CMS store, covering key concepts such as:  

- Building the **UI** entirely using **MudBlazor**.  
- **Integration** with **Authentication API** to understand **Guards & Layouts**.  
- **Integration** with an API to display various data, including:  
  - **Posts**  
  - **Post Details**  
  - **Categories**
